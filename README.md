Este repositório contém uma coleção de scripts destinados a praticar a manipulação de exceções em Python. 
O foco é aprender a tornar códigos mais robustos e evitar que o programa "quebre" inesperadamente.

Try / Except: Captura básica de erros.Except específicos: 
Tratamento diferenciado para erros como ValueError, e TypeError.Else / Finally: 
Execução de blocos de código após o sucesso ou obrigatoriamente ao final do processo.
Raise: Lançamento manual de exceções personalizadas.
